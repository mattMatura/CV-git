01 Technical Architecture

The Modern Data Stack (ELT)

Decoupling compute from storage. Data as code.

dbt
Incremental Models

Recomputing history is expensive. Use is_incremental() to process only new records based on timestamps.

⚠️ Strategic Value: If dashboards are slow/expensive, audit for unnecessary "full refreshes".

Schema Strategy
  • Star Schema (Kimball): Fact + Dimensions. Read-optimized. Rigid.
  • Activity Schema: Single wide time-series table. Entity + Activity + Timestamp. Enables flexible "time-travel" queries.

Advanced SQL for Products

02 Strategy & Physics

Hamilton Helmer's 7 Powers

Statics of persistent differential returns. Without one, you are a commodity.

1. Scale Economies

Unit costs decline as volume grows. High fixed, low marginal.

2. Network Economies

Value increases with usage. (NFX). Primary digital driver.

3. Counter-Positioning

New model incumbent cannot copy without self-damage.

4. Switching Costs

Operational entanglement. Value loss on exit.

5. Branding

Attribution of value to identical offering. Trust proxy.

6. Cornered Resource

Preferential access to scarcity (Talent, Patents).

Aggregation Theory

Power shifts to demand. Zero marginal cost.

  • Level 1: Supply Acquisition Netflix. Capital constrained.
  • Level 2: Transaction Cost Uber. Supply constrained.
  • Level 3: Zero Supply Cost Google/FB. Pure aggregation.
